# Flask微博主题舆情分析可视化系统 python796

## 项目概述

本项目是基于Python的Flask框架开发的微博主题舆情分析可视化系统，旨在通过数据爬取、统计分析和可视化展示，帮助用户更好地理解微博上的热点话题和舆情动态。

## 技术栈

- **编程语言**：Python
- **后端框架**：Flask
- **数据库**：MySQL
- **数据分析**：SnowNLP（情感分析）
- **可视化**：ECharts
- **前端技术**：HTML/CSS/JavaScript

## 功能模块

- **数据爬取**
- **用户登录注册**
- **功能统计**
  - 热词统计
  - 舆情统计
  - 文章统计分析
  - 发布地址统计
  - 评论统计
  - 情感分类统计

## 系统角色

- **用户**：可注册、登录，查看微博数据分析和可视化结果。

## 运行环境

- **Python环境**
- **数据库MySQL**
- **工具**：Navicat、PyCharm

## 核心亮点

- **实时数据爬取**：及时获取最新的微博数据。
- **情感分析**：利用SnowNLP进行情感分类统计，掌握舆情倾向。
- **可视化展示**：通过ECharts实现数据的直观展示。
- **系统稳定**：优化修复常见bug，保证系统稳定运行。
